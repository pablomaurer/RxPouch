# RxPouch

## Import
```js
import {RxPouch} from "RxPouch";
```

## replicate
[pouchdb](https://pouchdb.com/api.html#replication)

```js
RxPouch.replicate(source, target, [options])
```

## sync
[pouchdb](https://pouchdb.com/api.html#sync)

```js
let sync = RxPouch.sync(src, target, [options])
```

## plugin
[pouchdb](https://pouchdb.com/api.html#plugins)

```js
RxPouch.plugin(require('pouchdb-adapter-http'));
```

## debug
[pouchdb](https://pouchdb.com/api.html#debug_mode)

```js
RxPouch.debugEnable('*');
RxPouch.debugDisable();
```

## defaults
[pouchdb](https://pouchdb.com/api.html#defaults)

```js
RxPouch.defaults({
  option1: 'foo',
  option2: 'value'
});
```
